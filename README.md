# MinerProxy
ETH中转抽水软件，只支持ETH，软件绿色版。
图形化操作部署，可以加moongod85
可以提供阿里云国际代购，软件未抽水转向工具简单明了。
全网第一款连接加密软件，隐藏数据包本地部署，实现IP池本地化，可阻止被监管的目标
